Справка по графике:
  • Create(width, height, title, resizable) — создаёт окно, задаёт размер, заголовок и режим изменения размера.
  • Update() — обновляет изображение в окне и обрабатывает события Windows Forms.
  • Clear(r, g, b) — очищает буфер рисования цветом фона.
  • Lock() — блокирует bitmap для быстрого доступа к пикселям через память.
  • Unlock() — снимает блокировку bitmap после работы с пикселями.
  • SetPixelFast(x, y, r, g, b) — быстро устанавливает пиксель по координатам.
  • GetPixelFast(x, y) — возвращает цвет пикселя по координатам.
  • SetColor(r, g, b) — задаёт цвет обводки для фигур и текста.
  • SetFillColor(r, g, b) — задаёт цвет заливки для фигур.
  • Rectangle(x, y, width, height, thickness) — рисует прямоугольник только обводкой.
  • FillRectangle(x, y, width, height, thickness) — рисует прямоугольник с заливкой и обводкой.
  • Circle(x, y, radius, thickness) — рисует окружность.
  • FillCircle(x, y, radius, thickness) — рисует заполненную окружность.
  • Line(x1, y1, x2, y2, thickness) — рисует линию.
  • DrawText(text, x, y, size, fontName) — выводит текст на изображение.
  • DrawSprite(textureName, x, y) — рисует изображение из файла.
  • Delay(ms) — делает паузу на указанное число миллисекунд.
  • Close() — закрывает окно и освобождает ресурсы.
  • HideConsole() — скрывает консольное окно.
  • ShowConsole() — показывает консольное окно.
  • IsKeyPressed(key) — проверяет, зажата ли клавиша.
  • IsMousePressed(button) — проверяет, удерживается ли кнопка мыши.
  • IsMouseDown(button) — возвращает true один раз в момент нажатия мыши.
  • IsMouseUp(button) — возвращает true один раз в момент отпускания мыши.
  • IsMouseMove() — проверяет, было ли движение мыши.
  • MousePosition — возвращает текущую позицию курсора внутри окна.
  • MouseX / MouseY — текущие координаты курсора.
  • Width / Height — текущий размер рабочей области окна.
  • IsOpen — показывает, открыто ли окно приложения.
